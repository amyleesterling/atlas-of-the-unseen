# Fable — Preregistered predictions

Reconstructed from Fable's stated preregistration plan before implementation handoff.

1. Asymmetric memory interactions will produce pursuit spirals.
2. At least one self-haunting or Narcissus-like loop should emerge in a sufficiently long run.
3. A brush-severed pair should rebond faster than a never-met pair when the external trace remains intact.
4. Bleaching both the trace and internal memory should abolish that rebond advantage.
5. Low-dimensional signatures should produce visibly more false recognition than higher-dimensional signatures.
6. Consonance sonification should later be judged by reconstruction of phase-cluster count, frequency ratios, and merge timing—not by aesthetic resemblance alone.
