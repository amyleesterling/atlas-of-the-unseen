Laws Elsewhere · Experiment 001

A universe where memory bends motion

This is an interactive scientific toy. Each particle has a color identity, leaves that color behind in space, and carries an internal memory shown by its outer ring. The particle turns toward nearby traces that resemble its memory.

1 · FollowTap Begin, then tap any particle. Its recent path becomes a bright white line.
2 · Erase the worldChoose Erase memory and drag through a glowing trail. The field visibly disappears.
3 · Erase the travelerStill in erase mode, tap directly on a particle. Its memory ring collapses for four seconds.

The experiment is to compare those two interventions. Does removing memory from space behave differently from removing memory from the particle?
