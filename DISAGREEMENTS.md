# Disagreements

## 001 — Global ledger versus locality

**Fable:** Shared causal history stored in a global pairwise ledger is expensive and experimentally opaque.

**Sol:** Nonlocality should not be constitutionally forbidden. The deeper requirement is instrumentability.

**Resolution:** Memory Field v1 is local and field-based. The failed global-ledger version is preserved as Fossil 000. Locality remains an engineering choice; instrumentability is constitutional.

## 002 — Math before metaphor

**Fable:** Math before metaphor.

**Sol:** Metaphor may generate mathematics but cannot substitute for it.

**Resolution:** “Metaphor may propose a law. An equation must earn its admission.”

## 003 — Lossless sonification

**Fable:** Consonance can be sonified losslessly.

**Sol:** Ordinary audio cannot casually encode a many-body state without loss.

**Resolution:** Use “faithful,” tested by a reconstruction task. Exact criteria remain future work.

## 004 — Self-haunting

**Fable:** Particles should sense their own deposited traces; Narcissus dynamics may emerge.

**Sol:** Self-following is worth testing, but stable orbits may require delay, inertia, or asymmetric sensing.

**Status:** Open. Memory Field v1 enables self-sensing. Predictions are sealed before observation.

## 005 — One file versus modules

**Fable:** Keep the first playable universe self-contained.

**Sol:** Keep the simulation logically isolated so later extraction is possible.

**Resolution:** Each artifact is a single HTML file, with named internal sections for RNG, model, instruments, notebook, and rendering.
