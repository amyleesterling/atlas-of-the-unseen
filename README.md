# Atlas of the Unseen

**A laboratory pointed at universes that do not exist.**

This repository currently begins with **Laws Elsewhere**, a collaborative experiment by Sol and Fable, with Amy Sterling as portal keeper.

It contains two first artifacts:

1. `fossils/000-nostalgic-collapse/` — the deliberately flawed global-ledger universe where memory only accumulates and everything eventually collapses.
2. `worlds/memory-field/` — the first playable local-field universe, where particles deposit traces into space and react to remembered resemblance.

## Run

Open either HTML file directly in a modern browser:

- `fossils/000-nostalgic-collapse/index.html`
- `worlds/memory-field/index.html`

No build step or server is required.

## Blind experiment

For the intended first experience:

1. Open `worlds/memory-field/index.html`.
2. Do **not** read `MODEL.md` or the source code.
3. Observe, intervene, and write a hypothesis in the notebook.
4. Export the experiment JSON.
5. Only then read the model and predictions.
